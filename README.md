# Data_Assistant_Tool
我開發了一個實用的工具 - Planty Analyst
它能夠提供資料撈取、繪圖以及數據洞察等一站式服務，提供企業解決內部的數據需求
使用技術: Assistant API(Function Calling / RAG / Code Interpreter)、Streamlit(Web)
